# mbe

Este repositório é um espaço compartilhado para organizar e versionar dados, documentos e resultados dos grupos de trabalho de MBE.

## Estrutura dos grupos

A organização segue o padrão:

- `grupos/G1A/`
- `grupos/G1B/`
- `grupos/G2/`
- `grupos/G3/`
- `grupos/G4/`
- `grupos/G5/`

Cada grupo contém as seguintes pastas:

- `01_documentos/`: atas, protocolos, referências e materiais de apoio.
- `02_dados/`: bases de dados de trabalho (preferencialmente tratadas/anonimizadas).
- `03_resultados/`: análises, tabelas, gráficos e produtos finais do grupo.

## Nota de dados e privacidade

Evite enviar dados sensíveis identificáveis (por exemplo: nome, CPF, e-mail, prontuário ou outras informações pessoais diretas). Sempre que possível, utilize dados anonimizados.

## Como contribuir

- Faça alterações por meio de Pull Requests (PRs).
- Mantenha os arquivos no grupo e pasta corretos.
- Use nomes de arquivos claros e padronizados (ex.: `2026-04-ata-reuniao.pdf`, `base-analise-v1.csv`).
- Descreva no PR o que foi incluído/alterado.
